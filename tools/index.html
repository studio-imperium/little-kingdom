Kingdom Crushers Tools

Edits write directly to server/engine/assets/*.json. Reboot the game to pick up changes.

  • Animator — preview equipped characters and edit animations.json keyframes.
  • NPC Maker — edit npcs.json with friendly dropdowns for animations, projectiles and bombs.
  • Projectile / Bomb Maker — edit projectiles.json and bombs.json with a live preview.
  • Item Maker — edit items.json: stats, attack patterns, held/equipped visuals.
  • Spawn Maker — edit spawns.json: group, chance, NPCs and their offsets, with a live preview.
